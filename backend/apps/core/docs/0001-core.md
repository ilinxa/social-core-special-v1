# Core App - Initial Implementation

> **Version:** 0001
> **Status:** Complete
> **Date:** 2026-01-24

## Overview

The `apps/core/` app provides shared infrastructure for all Django applications in this project. It contains **no business logic** - only base classes, utilities, and common patterns.

---

## Components Implemented

### 1. Base Models (`models/base.py`)

| Model | Fields | Purpose |
|-------|--------|---------|
| `TimeStampedModel` | `created_at`, `updated_at` | Auto-timestamp tracking |
| `SoftDeleteModel` | `is_deleted`, `deleted_at`, `deleted_by` | Soft delete with manager |
| `UserStampedModel` | + `created_by`, `updated_by` | User attribution tracking |
| `UUIDModel` | `id` (UUID) | UUID primary key |
| `BaseModel` | TimeStamped + SoftDelete | Standard combination |
| `AuditModel` | UserStamped + SoftDelete | Full audit trail |

**Managers:**
- `SoftDeleteManager`: Excludes soft-deleted records by default
- `all_objects`: Access all records including deleted

**Usage:**
```python
from apps.core.models import TimeStampedModel, BaseModel

class Product(BaseModel):
    name = models.CharField(max_length=255)
```

### 2. Exceptions (`exceptions/`)

**Hierarchy:**
```
DomainException (base)
├── NotFound (404)
├── PermissionDenied (403)
├── ValidationError (400)
├── ConflictError (409)
├── AuthenticationError (401)
│   ├── InvalidCredentials
│   ├── TokenExpired
│   └── TokenInvalid
├── BusinessRuleViolation (400)
├── RateLimitExceeded (429)
└── ServiceUnavailable (503)
```

**Exception Handler:** Converts domain exceptions to consistent API responses:
```json
{
  "error": {
    "message": "Human-readable message",
    "code": "machine_readable_code",
    "details": {...}
  }
}
```

**Usage:**
```python
from apps.core.exceptions import NotFound, ValidationError

def get_product(product_id: int):
    product = Product.objects.filter(id=product_id).first()
    if not product:
        raise NotFound(resource="Product", resource_id=product_id)
    return product
```

### 3. Utilities (`utils/`)

#### JWT (`utils/jwt.py`)
| Function | Purpose |
|----------|---------|
| `encode_token(payload, expires_in)` | Create JWT |
| `decode_token(token)` | Validate and decode JWT |
| `is_token_expired(token, buffer)` | Check expiry |
| `get_token_expiry(token)` | Get expiry datetime |

#### Password (`utils/password.py`)
| Function | Purpose |
|----------|---------|
| `hash_password(password)` | Hash for storage |
| `verify_password(password, hashed)` | Verify match |
| `validate_password_strength(password)` | Get validation errors |
| `generate_temporary_password(length)` | Create random password |

#### DateTime (`utils/datetime.py`)
| Function | Purpose |
|----------|---------|
| `utc_now()` | Current UTC datetime |
| `to_user_timezone(dt, tz)` | Convert to user's timezone |
| `start_of_day(dt)` | Get 00:00:00 of day |
| `days_ago(n)` | Datetime N days ago |
| `format_iso(dt)` | Format as ISO 8601 |

---

### 4. Serializers (`serializers/`)

| Class | Purpose |
|-------|---------|
| `BaseInputSerializer` | Input validation (no create/update) |
| `BaseOutputSerializer` | Read-only model output |
| `EmptySerializer` | No body endpoints |
| `MessageSerializer` | `{"message": "..."}` responses |

---

### 5. Pagination (`pagination/`)

| Class | Page Size | Use Case |
|-------|-----------|----------|
| `StandardPagination` | 20 | Default for most endpoints |
| `SmallResultsPagination` | 10 | Dropdowns, autocomplete |
| `LargeResultsPagination` | 50 | Admin, exports |
| `CursorResultsPagination` | 20 | Large datasets, feeds |

---

### 6. Permissions (`permissions/`)

| Permission | Access Level |
|------------|--------------|
| `IsAuthenticated` | Logged in users |
| `IsAuthenticatedOrReadOnly` | Read: anyone, Write: authenticated |
| `IsStaff` | Staff users only |
| `IsOwner` | Resource owner only |
| `IsOwnerOrStaff` | Owner or staff |
| `IsVerified` | Email-verified users |
